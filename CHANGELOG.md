# 更新日志

## v1.1.0+2 - 2026-06-15

### 新增

- 新增歌曲宝请求节流和冷却机制，降低连续搜索、播放、下载时被拦截的概率。
- 新增本地音乐搜索。
- 新增本地音乐排序：最近下载、歌名、歌手。
- 新增本地歌曲信息编辑：歌名、歌手、歌词、封面图片路径或网址。
- 新增播放队列面板：查看队列、指定播放、拖动排序、移除歌曲、清空队列。
- 新增播放栏滚动歌词显示。
- 新增完整歌词面板，随播放进度自动高亮滚动。
- 新增音量记忆。
- 新增 Windows 下载目录选择。
- 新增 Windows 安装脚本模板。

### 改进

- 搜索结果不再自动请求详情页补封面，搜索速度更快，也更不容易触发歌曲宝拦截。
- 下载失败、播放失败和搜索失败提示更清楚。
- 已下载 MP3 会写入标题、歌手、歌词和封面 ID3 信息。
- 本地音乐封面优先从 MP3 内嵌封面提取，减少对网络封面的依赖。
- Windows 窗口标题和资源信息统一为“青听”。
- Windows 构建安装路径修正为 release 输出目录，不再写入 `C:\Program Files`。

### 修复

- 修复 Android 修改下载路径时可能触发 Flutter 依赖断言的问题。
- 修复播放栏底部溢出问题。
- 修复歌词解析残留 `<br />` 的问题。
- 修复部分 UTF-16 内嵌歌词读取异常。
- 修复歌曲宝搜索结果混入“网友刚刚下载/搜索”等动态内容的问题。
- 修复搜索页封面补全导致长时间卡住的问题。

### 签名

- v1.1 只更新版本号，release 签名应继续沿用原来的 `qingting-release.jks`。
- 普通版本升级不应更换签名，否则旧版 APK 无法覆盖升级。

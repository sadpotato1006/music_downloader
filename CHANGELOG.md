# 更新日志

## v1.1.5+7 - 2026-06-18

### 新增

- 本地页新增“随机”入口，可从扫描到的全部本地歌曲生成随机播放队列并立即播放，队列中可预知和移除后续歌曲。
- 本地歌曲列表显示专辑信息；扫描本地 MP3 时会读取 ID3 专辑，下载 MP3 时也会把专辑写入 ID3 标签。
- 歌词页升级为完整上滑页面，支持拖动进度条、自动高亮滚动，并可点击某句歌词直接跳转到对应时间。
- 设置页新增“启动软件时自动播放音乐”开关，可在启动时自动恢复播放队列中的当前歌曲。
- 设置页新增默认启动页面选项，可在搜索页和本地页之间选择，默认直达搜索页。
- 歌词页进度条下方新增五个控制按钮：随机/单曲循环切换、上一首、播放/暂停、下一首和歌曲信息。

### 改进

- 本地页随机播放、排序和搜索合并为单行工具栏；随机播放占主要宽度并改名为“点击开始随机播放”，排序和搜索保留图标式入口，按钮尺寸略微收小。
- 歌曲卡片支持点击任意位置播放并显示按压反馈，行内移除独立播放按钮，并缩小“下一首播放”按钮。
- 本地歌曲卡片取消歌名加粗和格式后缀，歌手和专辑改为同一行显示；封面占比略增，卡片间距略减，卡片宽度略增。
- 当播放器空闲时点击“下一首播放”会直接开始播放该歌曲。
- 下载、扫描、“下一首播放”等全局提示统一改为带青听图标的轻量通知，不再使用遮挡底部操作的黑色提示条。
- 扫描本地歌曲完成后也改用青听轻量通知，不再额外弹出底部黑色提示条。
- 播放队列里的歌曲卡片改成本地页相近样式，歌名不加粗，显示歌手和专辑；点击卡片任意位置即可跳转播放该歌曲，点击当前歌曲不会从头重播，并移除每行最右侧的拖动手柄。
- 歌词页播放方式默认随机播放，可通过左侧按钮一键切换为单曲循环。
- 底部播放栏只保留播放/暂停和播放队列按钮，不再显示播放进度条，减少对当前歌曲信息的遮挡。
- 顶部四个导航标签略微变薄并缩小文字，减少顶部占用空间。
- 本地歌曲扫描入口移到设置页。
- 同时下载默认值改为 1，用户调高时提示请求过频繁可能被网站拒绝。
- 设置页源代码链接最多两行完整换行显示，不再省略或出现多余换行。

### 修复

- 恢复本地音乐搜索，可按歌名、歌手、专辑检索本地列表。
- 移除启动时的整屏加载等待，应用打开后会先进入默认页面。
- 扩展歌曲宝详情页专辑解析，支持嵌套页面数据中的专辑字段，下载 MP3 时会继续写入 TALB 专辑标签。
- 扩展外部 MP3 歌词读取，支持 `USLT`、`SYLT`、歌词类 `TXXX` ID3 帧以及同名 `.lrc` 旁挂歌词文件。

### 改进

- 移除搜索、下载、本地、设置页面标题下方的说明/数量文本，减少顶部占用空间。

## v1.1.0+2 - 2026-06-15

### 新增

- 新增歌曲宝请求节流和冷却机制，降低连续搜索、播放、下载时被拦截的概率。
- 新增本地音乐搜索。
- 新增本地音乐排序：最近下载、歌名、歌手。
- 新增本地歌曲信息编辑：歌名、歌手、歌词、封面图片路径或网址。
- 新增播放队列面板：查看队列、指定播放、拖动排序、移除歌曲、清空队列。
- 新增播放栏滚动歌词显示。
- 新增完整歌词面板，随播放进度自动高亮滚动。
- 新增音量记忆。
- 新增 Windows 下载目录选择。
- 新增 Windows 安装脚本模板。

### 改进

- 搜索结果不再自动请求详情页补封面，搜索速度更快，也更不容易触发歌曲宝拦截。
- 下载失败、播放失败和搜索失败提示更清楚。
- 已下载 MP3 会写入标题、歌手、歌词和封面 ID3 信息。
- 本地音乐封面优先从 MP3 内嵌封面提取，减少对网络封面的依赖。
- Windows 窗口标题和资源信息统一为“青听”。
- Windows 构建安装路径修正为 release 输出目录，不再写入 `C:\Program Files`。

### 修复

- 修复 Android 修改下载路径时可能触发 Flutter 依赖断言的问题。
- 修复播放栏底部溢出问题。
- 修复歌词解析残留 `<br />` 的问题。
- 修复部分 UTF-16 内嵌歌词读取异常。
- 修复歌曲宝搜索结果混入“网友刚刚下载/搜索”等动态内容的问题。
- 修复搜索页封面补全导致长时间卡住的问题。

### 签名

- v1.1 只更新版本号，release 签名应继续沿用原来的 `qingting-release.jks`。
- 普通版本升级不应更换签名，否则旧版 APK 无法覆盖升级。
